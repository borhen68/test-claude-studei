## Basic Workflows

### Changing Layouts

1. Click **"Layouts"** tab (right toolbar)
2. Select a template (Hero, Duo, Trio, etc.)
3. Layout applies instantly
4. Photos redistribute automatically

### Editing Captions

1. Click **"Captions"** tab
2. Type your text
3. Choose font & size
4. Set alignment
5. See live preview below

### Adjusting Photos

1. Click any photo on canvas
2. **"Photo"** tab opens automatically
3. Adjust brightness/contrast
4. Rotate or apply filters
5. Click "Apply" to save

### Changing Themes

1. Click **"Themes"** tab
2. Preview color schemes
3. Click to apply
4. Use "Apply to All Pages" for consistency

## Drag & Drop

### Reorder Pages
- Drag page thumbnail in sidebar
- Drop at new position
- Pages renumber automatically

### Move Photos
- Drag photo slot on canvas
- Drop on another slot to swap
- Or drag from unused photos drawer

## Page Management

### Add Page
1. Click **"+ Add Page"** button (sidebar top)
2. New page appears at end
3. Navigate to it automatically

### Delete Page
1. Right-click page thumbnail
2. Select **"Delete Page"**
3. Confirm deletion

### Duplicate Page
1. Right-click page thumbnail
2. Select **"Duplicate Page"**
3. Copy appears below original

## Auto-Save

- Changes save automatically every 2 seconds
- Status shows in header: **ğŸ’¾ Saved 2s ago**
- Green dot = saved, spinning loader = saving
- Manual save: Click **"Save"** or press `âŒ˜S`

## Unused Photos

When you have photos not yet placed:

1. **Bottom drawer appears** automatically
2. Shows thumbnail gallery of unused photos
3. **Drag photos** onto canvas to add them
4. Or click **"Auto-fill Pages"** for smart placement

## Common Tasks

### Create Multi-Photo Page
1. Choose "Gallery" or "Quad" layout
2. Drag photos from unused drawer
3. Photos snap into slots
4. Adjust as needed

### Add Text to Page
1. Open "Captions" tab
2. Write your message
3. Style with fonts/sizes
4. Preview on canvas bottom

### Change Page Order
1. Drag page thumbnail up/down
2. Drop at new position
3. All page numbers update

### Export Final Book
1. Click **"Preview"** to review
2. Use viewer mode navigation
3. Click **"Export PDF"** when ready
4. Or **"Order Print"** to purchase

## Troubleshooting

**Changes not showing?**
- Check auto-save indicator
- Try manual save (âŒ˜S)
- Refresh if persistent

**Drag not working?**
- Ensure you're clicking photo, not background
- Try clicking and holding briefly before dragging

**Toolbar tabs not responding?**
- Some tabs require selection (e.g., Photo tab needs photo selected)
- Look for disabled state (grayed out)
